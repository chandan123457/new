## Animations & Transitions

### Keyframe Animations
1. `gradientShift` - Background gradient movement (15s)
2. `ripple` - Button click effect (0.6s)
3. `floating` - Dashboard cards floating (3s)
4. `slideInDown` - Alert appearance (0.5s)
5. `pulse` - Loading state (1.5s)
6. `fadeIn` - Modal overlay (0.3s)
7. `slideInUp` - Modal content (0.4s)
8. `badgePulse` - Badge emphasis (2s)

### Transition Effects
- Button hover: 0.3-0.4s ease
- Input focus: 0.3s ease
- Card hover: 0.3s ease
- Table row hover: 0.3s ease
- All interactions: Smooth, professional feel
